hexo安装问题

hexo部署到Github之后访问博客返回404的问题及其解决办法

问题描述：昨天在部署hexo的过程中，完成所有步骤之后对自己的博客地址进行访问出现了无法找到博客返回404页面的问题，仔细检查之后发现在创建github项目的时候项目名称应该创建为“用户名.github.io”的形式，而昨天在github上创建项目时创建了github用户名.github.io为名称的项目，创建项目的名称出错会直接影响到后续的部署，所以创建项目时一定要认真检查，并且区分大小写。

解决方法：
1.删除掉当前创建的项目，从新以“用户名.github.io”为名创建项目。
2.创建之后需要对_config.yml文件再次进行编辑，将repository中的改为新创建的项目的SSH。

3.初始化hexo博客，命令

hexo n “博客标题”

hexo g

hexo s

hexo d
该命令用于将博客部署到网络上，之后在地址栏输入用户名.github.io就可以看到写好的博客了
